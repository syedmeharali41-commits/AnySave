# bin/ Folder — yt-dlp.exe Setup

## Pehli Baar Setup (One-Time)

1. https://github.com/yt-dlp/yt-dlp/releases/latest par jayen
2. `yt-dlp.exe` (Windows) download karein
3. Is downloaded file ko **is hi folder** mein rakhein:
   ```
   omnisave/
   └── bin/
       └── yt-dlp.exe   ← yahan rakhein
   ```
4. App start karein — startup par automatically khud ko update kar lega

## Linux / macOS

- `yt-dlp` (no extension) download karein
- `chmod +x yt-dlp` command chalayein
- `bin/` folder mein rakhein

## Auto-Update

App start hone par `yt-dlp -U` automatically run hota hai.
Koi Python install karne ki zaroorat nahi.
